# Java Projects

A collection of Java projects from my Software Engineering studies, covering compiler construction, data structures, and algorithms.

## Projects

- **MidiJava** — Custom compiler for a simplified Java language, supporting variables, arithmetic, if/else, and while loops, compiled to a custom bytecode and executed on a virtual machine.

- **OptimizingCompiler** — Extension of MidiJava that builds an abstract syntax tree (AST) for expressions and applies optimizations like constant folding and elimination of redundant operations before generating bytecode.

- **WordCounter** — Word frequency counter implemented with three different data structures: binary search tree, hash table with chaining, and hash table with open addressing. Includes a performance comparison.

- **TouristGraph** — Weighted directed graph implementation that finds the optimal route between two locations, balancing distance and scenic beauty, using a Dijkstra-based algorithm.

- **SudokuSolver** — Solves 9x9 Sudoku puzzles using the backtracking algorithm.

## Tech

All projects are written in plain Java with no external dependencies.
